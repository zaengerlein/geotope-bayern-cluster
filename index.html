Cluster-Kriterien

Die Zuordnung basiert aktuell auf einer ersten Keyword-Heuristik über Namen und Beschreibung. Die Struktur ist nun so gebaut, dass du das JSON später leicht austauschen oder fachlich bereinigen kannst.

Prozess

Glazial, fluvial, karstisch, vulkanisch/magmatisch, Impakt, Sedimentation, Fossilführung.

Morphologie

Klamm, Schlucht, Felsformation, Höhle, Moor, See, Steinbruch, Talform.

Erweiterbarkeit

Das importierte JSON kann später um Hauptprozess, Region, Gesteinstyp oder Tourenrelevanz ergänzt werden.

Cluster auf einen Blick

Die Zahlen werden direkt aus der JSON-Datei berechnet. Mehrfachzuordnungen bleiben sichtbar, weil ein Geotop mehreren Aspekten gleichzeitig zugeordnet sein kann.

Datensatz

Die Tabelle wird beim Laden der Seite aus der importierten JSON-Datei aufgebaut. Über das Suchfeld kannst du direkt nach Namen, Beschreibung oder Cluster filtern.
